### express mongoose
    使用nodejs + mongodb 快速构建项目,前端使用基于bootstrap风格的ui，可用于各种场景的web风格。
#### 安装部署
    nodeJs（v8.0以上） 、mongodb(3.4)、
    
```
 1. 安装 `Node.js[必须]` `MongoDB[必须]``
 2. 启动mongodb
 3. install 依赖包
 4. 启动npm run （dev | prod）启动项目
 5. 访问地址http://localhost:8888
```
